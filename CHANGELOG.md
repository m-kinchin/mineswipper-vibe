# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Features
- Classic Minesweeper gameplay
- Multiple difficulty levels (Beginner, Master, Expert, Custom)
- Theme support (Modern Light, Dark Mode, Classic Win98)
- Adjustable cell sizes for accessibility
- Game state persistence
- Settings modal with version display
